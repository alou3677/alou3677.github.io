a run-through of my work experience and personal projects

SAP Labs
Software QA Engineer: 5.16 - Present | Project Manager Intern: 6.17 - Present

Starting in May of 2016 after finishing my freshman year at Cal, I was interning as a software engineer for SAP Labs in Palo Alto, California. The overall description of the job was to help develop and update the testing framework that the QA team already had in place for SAP's Business Objects Information Steward web application.


When I joined, 99% of the team's testing framework was based on Java code that was called upon via shell scripts to send requests to an Information Steward server, allowing us to execute a large number of testcases with ease.


My tasks consisted of updating "goldlog" files with which we compared test run outputs against to verify expected behavior throughout patches and new releases and creating the shell scripts that would run a suite of testcases.


However, the most powerful tool I've learned to use at SAP is Apache JMeter. While built for load lesting web applications, JMeter is also extremely useful for the automation of web requests. Using JMeter, I have been able to expand our team's testing framework at a much faster rate with JMeter's built-in templates and increase our testing performance for JMeter-based testcases by up to 50% with less web requests and high-performance JSON output parsing.
